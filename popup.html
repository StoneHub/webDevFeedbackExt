Quick Start
  1. Pick Element for clickable DOM capture or Region for PDFs and screenshots.
  2. Use the primary button to start the selected workflow on the current tab.
  3. Region capture opens a separate editor tab where you draw the box and add the note.
  4. Re-open the page later to re-export JSON, Markdown, or the AI prompt from the history panel.
Keyboard Shortcut
Toggle element mode Ctrl+Shift+F
Local `file://` PDFs may require enabling "Allow access to file URLs" for this extension.
