Dev Feedback Capture

Annotate web elements on localhost

⚠️ This extension only works on localhost pages
Quick Start
  1. Navigate to a localhost page
  2. Click the floating button or press Alt+F
  3. Hover and click elements to capture feedback
  4. Export as JSON or Markdown
